# shoppingonline_DoAnWebBanHoa

Đây là đồ án xây dựng **website bán hoa online** gồm 3 phần chính:

- **client-admin**: giao diện quản trị
- **client-customer**: giao diện người dùng/khách hàng
- **server**: backend xử lý API, kết nối database MongoDB

## Công nghệ sử dụng

### Frontend
- ReactJS
- HTML, CSS, JavaScript

### Backend
- Node.js
- ExpressJS

### Database
- MongoDB
